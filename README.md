# YardiGo 🏷️

Nederlandse garage sale & rommelmarkt discovery app.

Vind rommelmarkten, garagesales en kofferbakverkopen bij jou in de buurt.

## Tech Stack

- **Frontend:** Vanilla HTML/CSS/JavaScript (PWA)
- **Kaarten:** Leaflet + OpenStreetMap
- **Backend:** Supabase (PostgreSQL)
- **Hosting:** Vercel

## Lokaal draaien

Open `index.html` in je browser, of gebruik een lokale server:

```bash
npx serve .
```
