🏅 Badge-systeem — Supabase setup

Het badge-systeem slaat handmatige badges op in de profiles tabel. Voer dit eenmalig uit in de Supabase SQL Editor:

ALTER TABLE profiles ADD COLUMN IF NOT EXISTS admin_badges text DEFAULT '[]';
ALTER TABLE profiles ADD COLUMN IF NOT EXISTS is_blocked boolean DEFAULT false;
→ Open SQL Editor
